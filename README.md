## AI 기반 여행 일정 추천 시스템 - 프론트엔드 개발

### 기술 스택 (Frontend)
React 

## 프론트엔드 구현 기능

1. 회원가입 및 로그인 
- `SignupPage.jsx` → 회원가입 폼 구현 및 API 연동
- `LoginPage.jsx` → 로그인 API 연동 
- `AuthContext.js` → `useContext`로 로그인 상태 전역 관리


2. 여행 일정 추천
- 사용자 입력값을 받아 AI API 요청 후 일정 추천

3. 여행 추천 결과 페이지 
- AI가 생성한 여행 일정을 카드 형식으로 표시


4. 프로필 조회 (사용자 정보 확인)
- 로그인한 사용자의 프로필을 API를 통해 가져옴
